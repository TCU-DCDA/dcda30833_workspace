# Live-Coding Plans

Per-session teaching plans for the **hands-on technique weeks**, taught by
building a sketch *with the class* in the p5.js Web Editor rather than from a
slide deck. (Conceptual framing weeks use the decks in `slide_decks/`.)

Each plan is built around **what you build live**, not bullet points, and
follows the course rhythm: open → live build → discussion → live build →
studio → exit. The goal on screen is discovery — predict before running, keep
instructive mistakes, and re-run to show the *range* one system produces.
