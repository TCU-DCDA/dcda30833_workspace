# Labs

Weekly technique-focused exercises for DCDA 30833.

Each lab isolates a single concept and includes:
- `index.html` — HTML wrapper to run your sketch
- `sketch.js` — Your p5.js code (or `style.css` for Lab 0)
- `reflection.md` — 3-5 sentence response to the reflection prompts

## Lab List

| Lab | Title | Concept |
|-----|-------|---------|
| 00 | Structure & Surface | HTML/CSS as systems |
| 01 | Rule Without Random | Deterministic emergence |
| 02 | Random Without Design | Pure chaos |
| 03 | Controlled Randomness | Taming chance |
| 04 | One-Hue World | Color as constraint |
| 05 | Palette as Prison | Limited color systems |
| 06 | Text as Object | Text as visual material |
| 07 | Prompt as Material | AI prompting as constraint |
| 08 | Envelope Only | Sound shape without pitch |
| 09 | Rhythm Without Notes | Temporal pattern |
| 10 | Mapping Without Message | Data → behavior |
| 11 | Three Mappings | Same data, different systems |
| 12 | System Translation | Identity across rewrites |
| 13 | Failure Harvest | Productive glitch |

## Running Labs

Open any `index.html` in a browser, or use VS Code's Live Server extension.
