PROJECT 2 --- Generative Visual Study

Course: DCDA 30833 --- Creative Coding for the Humanities
Due: Week 4 (Thursday)
Tools: p5.js
Submission: GitHub link to your project folder + short written reflection

Purpose

This project synthesizes:

  • LAB 1 --- Rule Without Random
  • LAB 2 --- Random Without Design
  • LAB 3 --- Controlled Randomness

You will design a generative visual world governed by:

  • rules
  • chance
  • constraint

Core Requirements

  • Must be a generative system
  • Must use:
    • randomness
    • probability
    • constraints
  • Must show variation across time or refresh
  • Must maintain a clear system identity

Technical Requirements

  • Use p5.js
  • Default canvas: 400 × 400
  • No images
  • No external data

Reflection (5--7 Sentences)

What rule defines the system?
Where does randomness enter?
What constraint mattered most?

Course Arc

This project establishes full generative behavior design.
