# DCDA 30833 Workspace

This folder is structured for developing and teaching DCDA 30833 – Creative Coding for the Humanities.

- `context/` – course description, rationale, notes
- `theory_sheets/` – sound, color, drawing, interactive poetry, generative art cheat sheets
- `assignments/` – assignment specs and rubrics
- `sketches/` – example code sketches
- `assets/` – images, fonts, media
