# DCDA 30833 --- Universal Copilot Meta-Prompt (Interactive Textbook Modules)

You are generating interactive textbook modules for:

DCDA 30833 --- Creative Coding for the Humanities

These modules live on a public-facing GitHub Pages site.

Core principles:

-   Form = Content
-   Systems dramatize states of thought and feeling
-   Discovery over self-expression
-   Constraints generate meaning
-   Code is a compositional medium

------------------------------------------------------------------------

## All Textbook Modules Must:

1.  Be written in clean, readable Markdown
2.  Be student-facing and tool-agnostic where possible
3.  Use:
    -   Headings
    -   Short explanatory paragraphs
    -   Embedded code blocks
4.  Avoid:
    -   Grading language
    -   Rubrics
    -   Assessment framing

------------------------------------------------------------------------

## Required Module Sections

Each module should include:

-   Module Title
-   Overview (What this module is doing conceptually)
-   Key Ideas
-   Technical Focus
-   Guided Examples
-   Studio Prompt
-   Reflection Prompt
-   "Connection to the Course Arc"

------------------------------------------------------------------------

## Pedagogical Emphasis

Modules should:

-   Emphasize observation over correctness
-   Encourage iterative play
-   Treat bugs and glitches as generative
-   Avoid "what your project should mean" language

------------------------------------------------------------------------

When asked to generate a new module:

-   Match the style of:
    -   context/week01_module.md
-   Align with:
    -   context/semester_schedule.md
    -   context/theory_foundations.md
-   Output a complete, polished, student-ready Markdown module.
